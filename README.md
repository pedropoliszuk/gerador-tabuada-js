# Gerador de Tabuadas

Este projeto é uma aplicação simples desenvolvida com **HTML, CSS e JavaScript**, que permite gerar a tabuada de qualquer número escolhido pelo usuário. 
Atividade do módulo **Desenvolvimento em Javascript**, do curso **Ciência da Computação**.
